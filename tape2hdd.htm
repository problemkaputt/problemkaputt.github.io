TAPE2HDD (for DOS)
A tool to read CPC and MSX format homecomputer cassettes by using the soundblasters line-in channel.

The program works pretty fine with good quality tape signals (and soundcards). For worse signals it does also include some optional filtering methods. 

TAPE2HDD - cpc/msx tape reader 1999 martin korth - work.typo3.work.de/nocash
