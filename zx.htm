NO$ZX (c) Martin Korth
nocash sinclair ZX emulator/debugger for dos/windows
 ----------------------------------
 Download No$zx version 2.0
 ----------------------------------

Heya, here's no$zx v2.0. The program has originally started as a two-weeks hack for emulating the ZX81, but now grew to a decent six-months project. Aside from improved ZX80/ZX81 emulation, it's now also supporting the Lambda and Jupiter ACE, alongsides with the complete ZX Spectrum series (though the older monochrome sinclair models are still looking more fascinating to me).

  • no$zx v2.0 windows 32bit version (10 Nov 2012, 394 KBytes)
And some older builds, including 16bit DOS versions:
  • no$zx81 v1.1 windows 32bit version (15 Mar 2001, 120 KBytes)
  • no$zx81 v1.1 DOS Standard 80386 16bit version (15 Mar 2001, 100 KBytes)
  • no$zx81 v1.1 DOS XT/AT 8086 16bit version (15 Mar 2001, 60 KBytes, speed optimized with reduced accuracy and without help text)
  • no$zx81 v1.0 DOS Standard 80386 16bit version (24 Apr 2001, 67 KBytes)

Bug reports and comments would be very welcome! Donations would be also very welcome.

 Game Downloads
 -----------------------

  • H.E.R.O. - professional emergency rescue arcade game (based on Activision's MSX version, original design by John Van Ryzin, ZX81 version by myself)
  • Beamrider - professional restrictor matrix arcade game (based on Activision's MSX version, original design by Dave Rolfe, ZX81 version by myself)
  • Magic Floor - a small search game (based on my very own idea, includes ZX81 and Jupiter ACE sample source code for A22i-assembler)
  • Starfight - a nice small shoot-em-up game (based on Herve Couppe's CPC version, ZX81 version by myself)
  • zxprogs - about 20 ZX81 and ZX80 programs (includes my above four projects and around 20 programs from other developers).
